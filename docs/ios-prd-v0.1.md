## 1. 타겟 사용자 (페르소나 1명)

- **이름(프록시)**: 권오영
- **환경**: macOS + iPhone 17 Pro. macOS Calen을 데스크에서 주력으로 사용. iPhone은 이동 중/회의 중 참조.
- **성향**: ADHD — 맥락 전환 비용 큼. 작은 마찰에 작업 포기. 시각적 잡음에 민감.
- **기존 습관**: Google Calendar가 모든 일정의 single source of truth. 하루 10~20회 확인.
- **고통**: "지금 뭐해야 하지?" / "다음 일정 언제?" / "이 메모를 어떻게 일정으로 바꾸지?" 세 가지 마찰이 지속 발생.

## 2. iOS Calen의 핵심 가치 (macOS와 구별됨)

**macOS Calen**은 "계획·회고·AI 대화"의 깊은 작업 환경.
**iOS Calen**은 다음 3가지에만 집중:

1. **즉시 확인** — 지금/오늘/이번 주 일정을 0.5초 안에 인식 (가독성이 전부).
2. **가벼운 조작** — 터치 2회 이내로 이벤트 생성·시간 이동·완료 체크.
3. **맥락 복원** — 외부 앱에서 본 내용(메일·메모·채팅)을 캘린더로 빠르게 흡수.

**iOS는 macOS의 미러가 아님**. 무거운 AI 대화, 긴 회고 작성, 복잡한 Goal 설정은 iOS에서 하지 않음. 그것들은 desktop 작업.

## 3. JTBD (5개)

- JTBD-1: "오늘 뭐가 있더라?" 확인 → **오늘 탭 첫 화면** 0.5초 표시.
- JTBD-2: "다음 회의까지 몇 분 남았지?" → **위젯 + 상단 배너**로 앱 안 열어도 확인.
- JTBD-3: "10시 미팅이 11시로 바뀌었네" → **이벤트 탭 → 시간 하나 바꾸기**, 왕복 3초 이내.
- JTBD-4: "이 메일 내용을 일정으로" → **공유 시트 → Calen → 프리필**, 왕복 5초.
- JTBD-5: "내 일정이 Mac에도 보이게" → Google Calendar 싱크 = 자동 (설명 불필요).

### 인터랙션 스펙

#### 월 그리드에서 블록 조작 (사용자 요청 반영)
- **탭 1회**: 해당 **하루 Day Sheet** 오픈 (JTBD-1).
- **블록(이벤트 바) 탭 1회**: 즉시 **편집 시트** 오픈 (현재 Day Sheet 경유 2단계 → 1단계로 단축).
- **블록 long-press(0.4초)**: "이동 모드" 진입 — 블록이 살짝 들뜨고 진동(haptic). 드래그로 **날짜 셀 간** 이동. 손 떼면 확정.
  - 같은 날 시간 변경은 v0.1.0 범위 밖(Day Sheet 편집으로 처리).
- **+FAB**: 선택된 날짜의 생성 시트.

#### Day Detail Sheet
- `.sheet(presentationDetents: [.medium, .large])` — 처음 medium, 스와이프로 large.
- 상단: `< 4월 20일 월요일 >` + 좌우 chevron (이전/다음 날).
- 바디: ScrollView + 이벤트 카드 리스트 (시간 오름차순).
  - 카드: 좌측 3pt 카테고리 색 bar + 제목 16pt semibold + 시간 13pt secondary + 위치 12pt tertiary.
  - 탭 → `EventEditSheet`.
  - 스와이프 왼쪽 → 삭제 confirm.
- 빈 상태: "일정이 없어요" 14pt + FAB hint.
- 가로 스와이프로 **이전/다음 날** 이동 (주/월 경계 넘어감).

## 5. 품질 기준 (비기능 요구사항)

- **가독성**: 한글 기준 이벤트 제목 최소 **4글자** 완독 (말줄임 허용 최대 1자).
- **반응성**: 탭/스와이프 즉각 시각 피드백 (<100ms).
- **성공 지표 (나 자신 기준)**:
  - 아침 출근 지하철에서 "오늘 뭐 있지?" 10초 내 답 획득.
  - 갑작스런 일정 변경 15초 내 저장.
- **Dynamic Type**: 최소한 `.large` 기준까지는 레이아웃 깨지지 않음 (접근성 XL은 v0.2).
- **다크 모드**: 지원.
- **Offline**: 마지막 fetch된 이벤트는 표시, 편집은 online 한정.

## 6. 성공 = 언제 이 MVP를 "완성" 이라 부르나

다음 5가지 중 **5개 모두** 통과:
1. 월 그리드에서 오늘 포함 6주가 한 화면에 깨짐 없이 표시.
2. 날짜 탭 → Day Sheet 열림 → 이벤트 탭 → 편집 시트 → 저장까지 **3초** 이내 왕복.
3. 이벤트 long-press → 다른 날짜 드롭 → Google Calendar에 PATCH 반영 (refetch 시 새 위치).
4. Google 로그인 없이도 앱 crash 없이 "로그인 필요" 빈 상태 보임.
5. 다크 모드에서 모든 텍스트 WCAG AA 대비 통과.

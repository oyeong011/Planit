App functionality

One small place for calendars, tasks, and planning.

01

Unified calendar view.

Calen shows Google Calendar and Apple Calendar events together in a native macOS menu bar workspace.

02

Event planning actions.

Users can create, edit, reschedule, and delete calendar events after reviewing the action in the app.

03

AI-assisted review.

Calen can pass selected calendar context to the user's local Claude or Codex CLI to suggest plans.

Google data use

Calendar access is used only to run Calen's calendar features.

Calen is a desktop client. It syncs directly with Google APIs from the user's Mac and does not operate a Calen cloud service that receives calendar data.

Read the Privacy Policy
calendar.events

Used to display Google Calendar events and let the user create, edit, move, or delete events inside Calen.

calendar.calendarlist.readonly

Used to list the user's calendars, show calendar names, and let users choose which calendars are visible or sensitive.

userinfo.email

Used to show which Google account is signed in and to store OAuth tokens under the right local Keychain account.

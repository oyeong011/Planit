# Calen

Menu bar calendar planning for macOS with Google Calendar, Apple Calendar, Reminders, and local Claude/Codex CLI suggestions.

```bash
brew install --cask oyeong011/calen/calen-ai
```

> Homebrew/homebrew-cask registration is not complete yet. Calen installs through the project tap while it works toward Homebrew's notable requirements.

## What it does

| Plan | Sync | Assistant | Updates |
| --- | --- | --- | --- |
| Find free slots, move unfinished tasks, drag events | Google Calendar, Apple Calendar, and Reminders | Uses your local Claude Code or Codex CLI | Sparkle updates |

Example workflows: find a 30-minute Monday slot for a long-term goal, or move unfinished tasks into tomorrow's open blocks.

Calen is early stage. Recurring edits are limited, some UI states need polish, and feedback is welcome.

Compared with Fantastical or BusyCal, Calen is smaller: menu bar planning plus local CLI scheduling help, not a full calendar-suite replacement.

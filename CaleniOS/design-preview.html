새 일정 추가 (Bottom Sheet)

4 · v0.5 → v0.6 핵심 변경

실제 시안 33장 기준 정확 반영
❌ 카드 4종 다른 색 → ✅ 모두 라이트 블루 톤 + 저녁식사만 살짝 핑크 (시안 정확)
❌ 마이크 floating → ✅ inline 5탭 (Home/Cal/Mic/Person/Gear)
❌ 영어 카피 → ✅ 한국어 (기상 / 직장 / 거래처 회의 / 저녁식사 / 오늘의 일정)
❌ "Today's Schedule" 큰 헤더 → ✅ "오늘의 일정" outline 둥근 라벨
❌ 카드 내용 체크박스+칩 → ✅ 일정 요약 1줄 + 라이트 블루 액션 칩
❌ Calendar는 빈 화면 → ✅ 월간 + multi-day "호주 출장" 막대 + 하단 시간순 리스트
✅ Ai Call: 큰 마이크 orb + 글로우 + "최근 추가" / "추천 일정" 회색 카드
✅ Setting: 큰 프로필 사진 + 펜 아이콘 + 사용자 정보 / 설정 key-value 리스트
✅ Login: "나만의 AI 비서," + 큰 Calen 로고 + 라이트 블루 그라디언트 + 캡슐 CTA
✅ 새 일정 sheet: 둥근 시트 + 미니 달력 + 시간 wheel + 하단 음성 바
다음 단계
Sprint A 빌드 ✅ 완료 (CalenTheme / Theme tokens / CalendarEventV1 / CloudKitSyncCoordinator).
Sprint B (iPad SplitView) 또는 시안에 맞춘 HomeView 리뉴얼로 넘어갈 예정.
